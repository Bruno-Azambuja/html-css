Adicionando cores

Neste exercício usaremos o estilo inline, que é o estilo menos usado pois deixa o código misturado html e css, causando confusão na hora de identificar as coisas.

O estilo inline é usado dentro da tag que deseja aplicar o estilo, no caso se aplicar um estilo em todos os "h1" caso fizer alguma alteração, deverá fazer em todos os outros "h1". Por conta disso ele é usado apenas em casos específicos.
