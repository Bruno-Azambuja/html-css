Exemplos de tags semanticas e não semanticas

Negrito / Destaque

Nessa frase temos um terno em negrito usando a tag <b> (não semântica).

Aqui temos um termo em destaque usando a tag <strong> que é (semântica).

Itálico / Ênfase

Nessa frase temos um termo em itálico isando a tag <i> (não semanticas).

Nessa temos um termo em destaque usando a tag <em> (semântica).


Podemos criar também um texto marcado com a tag <mark> (semantica).

Aqui temos outro texto marcado mas com a cor diferente, pois foi usado CSS.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno.
A tag <big>, usada para o texto grande, está obsoleta no html5, pois para aumentar um texto usa-se o CSS. (não semantica)
A a tag <small> não esta obsoleta. O professor usa um exemplo:
Nos contratos "leias as letras pequenas", quer dizer que elas são observações importantes mas você consegue dar sentido para letras pequenas com a tag <small> (semântica).

Texto Deletado

Podemos marcar um texto excluído com a tag <del>, para considerar que o texto deve ser lido mas não considerado (semântica).

Texto Inserido

Podemos marcar um texto como inserido com a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Também temos o texto sublinhado com a tag <u>, que é (não semantica).

Texto Sobrescrito

Para inserir coisas do tipo X20. No X elevado a 20, o 20 deve estar na tag <sup> (semântica).

Texto Subescrito

Para inserir coisas do tipo H2O. O "2" do H2O deve estar na tag <sub> (semântica).
