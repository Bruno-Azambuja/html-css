Neste estilo o CSS é aplicado dentro da tag style que deve ser aberta dento do head (cabeçalho do código). Geralmente aberto na última linha abaixo do title.
Esta maneira permite usar o css separado do html melhorando a visualização.

Mas ainda sim, não é o estilo mais usado.

Os títulos "h1" destacados de vermelho acima, estão aplicado o css apenas nos "h1", com isso todo h1 que eu criar, esse estilo será aplicado.
