Usando Links Externos

Para links, usamos a tag <a> (com fechamento). Com essa tag tem alguns atributos para melhorar a navegação.

O atributo (target =""), informa ao navegador como o redirecionamento deverá ocorrer, abrindo a página na mesma janela/aba do navegador ou abrindo uma nova janela/aba.

Os atributos target são:

  • _blank: abre a página em uma nova janela/aba;
  • _self: abre a página na mesma janela;
  • _parent: abre a página na mesma janela do link;
  • _top: cancela todos os demais frames e abre a nova página no mesmo navegador.

Seguido do atributo (rel=""), que não encontrei ao certo sua utilização na ancora <a>, mas de acordo com as aulas, usamos:

  • rel = "external"- Se for abrir em outra janela/aba.
  • rel = "next" - Se for abrir na mesma janela/aba.
  • rel = "prev" - Se for voltar para pagina principal(home, index..).
  • rel = "nofollow" - O nofollow é um atributo HTML que permite pedir ao Google e a outros motores de busca a desconsideração de um link em específico para fins de SEO. De forma simples, como o nome sugere, ele informa os bots de rastreamento do Google para não “seguirem” o link.
