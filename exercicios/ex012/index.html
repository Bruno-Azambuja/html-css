Inserindo Vídeos Hospedados Localmente

Este vídeo está hospedado no meu próprio servidor.

Adicione o vídeo com a tag vídeo, adicione o atributo "controls" para aparecer os controles de play/pause e adicione o atributo "poster" caso queira colocar uma imagem no vídeo que ficará exposta quando o vídeo ainda não foi executado.

Também pode ser usado vários outros atributos como "width" para reduzir o tamanho do quadro do vídeo na tela.
