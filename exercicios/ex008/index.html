Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno.
A tag <big>, usada para o texto grande, está obsoleta no html5, pois para aumentar um texto usa-se o CSS. (não semantica)
A a tag <small> não esta obsoleta. O professor usa um exemplo:

Texto Deletado

Podemos marcar um texto excluído com a tag <del>, para considerar que o texto deve ser lido mas não considerado (semântica).

Texto Inserido

Podemos marcar um texto como inserido com a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Também temos o texto sublinhado com a tag <u>, que é (não semantica).

Texto Sobrescrito

Para inserir coisas do tipo X20. No X elevado a 20, o 20 deve estar na tag <sup> (semântica).

Texto Subescrito

Para inserir coisas do tipo H2O. O "2" do H2O deve estar na tag <sub> (semântica).

Outras Formatações

Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Ela está sendo apresentada com uma fonte mono espaçada, usando a tag < code > que são todas as letras com o mesmo espaço, tornando melhor a leitura. O valor semântico que ela representa é sua principal vantagem, indicando ao navegador que se trata de um código de computador (fonte de código).

Exemplo de Código Phyton

(Essa linha de código abaixo, está sob a tag code).

num = int(input('Digite um número')) if num % 2 = 0: print(f'O número {num} é PAR') else: print(f'O número {num é ÍMPAR}) print('Fim do programa')

Para deixar identado, envelope todo o conteúdo junto com a tag code e coloque na tag < pre >.
Ficará assim: 

        
            num = int(input('Digite um número')) 
            if num % 2 = 0:
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num é ÍMPAR})
            print('Fim do programa')

Texto Invertido

Com a tag (bdo) seguido de (rtl), você cria um texto invertido veja o exemplo: Estou aprendendo criar conteúdo em HTML.

Com a tag "bdo" aparece para escolher a "dir", que seria a direção, daí tem a opção "rtl"(right to left) - direita para esquerda, que você coloca entre as aspas do dir.
Isso fará o texto invertido.
