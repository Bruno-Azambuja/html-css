font-size:

Medidas Absolutas

cm, mm, in(polegada), px, pt(ponto), pc(paica).

pt(ponto), pc(paica) = São medidas da tipografia de papel. Não se recomenda utilizar em documento impresso em tela.

Medidas Relativas

em, ex, rem, vw, vh, %.

  • vp = (view port = É o tamanho da sua tela).
  • em = Medida relativa ao tamanho atual da fonte(em maiúsculo).
  • ex = Medida relativa a altura x de uma fonte.
  • rem = É tipo o (em), mas é relativo a fonte que está configurada no body.
  • vw = (view width = Largura da view port).
  • vh = (view height = Altura da view port).

Para a medida de fonte no font-size, ele recomenda-se o (px) e o (em).
Normalmente os navegadores colocam o font-size com 16px. Se vc não colocar nada vai estar em tamanho padrão 16px.

(16px é igual a 1em).
(2em é igual a 32px). etc.

No final tente usar mais o (em).
