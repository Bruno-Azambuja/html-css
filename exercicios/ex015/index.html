Estilo Externo

O estilo externo é mais usado para aplicação do css. Você cria a tag link:css na área do head em baixo do title e aperta Ctrl + clique para criar o arquivo css e faz o estilo separado.

Para aplicar o css em outras páginas, basta apenas copiar o link criado em head para as outras páginas.
